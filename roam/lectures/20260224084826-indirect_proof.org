#+title: Indirect Proof
#+filetags: :lecture:
* Lecture Notes
To prove a statement \(P\) by contradiction, we suppose \(\neg P\), get a contradiction and conclude \(P\).

We likely use this type of proof when:
1. What we prove means "\(P\), not \(Q\)"
2. We fail to prove it directly

When we prove a number is irrational or a set is empty, it should be proven by contradiction.

* Examples
** Example 1
\(\forall n \in \mathbb{Z}\), if \(n^{2}\) is even, then \(n\) is even.
Proof. Suppose \(n\) is an integer.
Suppose that \(n^{2}\) is even, that is \(n^{2}=2k\) for some integer \(k\).
We prove by contradiction that \(n\) is even.
Suppose \(n\) is odd, then \(n=2k+1\) for some integer \(k\) and so \(n^{2}={(2k+1)}^{2}=4k^{2}+4k+1=2{(2k^{2}+2k)}+1\) where \(2k^{2}+2k\) is an integer.
Thus, \(n^{2}\) is odd which contradicts the assumption that \(n^{2}\) is even.
Hence, \(n\) is even.
