* Examples
** Example 1
$20 \bmod 3 = 2$ because $20 = 6 \times 3 + 2 \quad \text{and} \quad 0 \le 2 < 3$.

** Example 2
Prove that for all integers $n$, $n^3 + n$ is divisible by 3.

Proof: Suppose $n \in \mathbb{Z}$. By the quotient–remainder theorem, there are three possible cases.

*** Case 1: $n = 3q$
$n^3 + n = (3q)^3 + 3q = 27q^3 + 3q = 3(9q^3 + q)$.

Since $9q^3 + q \in \mathbb{Z}$, we conclude $3 \mid (n^3 + n)$.

*** Case 2: $n = 3q + 1$
\begin{align*}
n^3 + n &= (3q+1)^3 + (3q+1) \\
        &= 27q^3 + 27q^2 + 9q + 1 + 3q + 1 \\
        &= 27q^3 + 27q^2 + 12q + 2 \\
        &= 3(9q^3 + 9q^2 + 4q) + 2.
\end{align*}

But we only need divisibility mod 3. Compute mod 3:

$n \equiv 1 \pmod{3} \Rightarrow n^3 \equiv 1 \pmod{3}$.

So, $n^3 + n \equiv 1 + 1 = 2 \equiv 0 \pmod{3}$.

*** Case 3: $n = 3q + 2$
$n \equiv 2 \pmod{3} \Rightarrow n^3 \equiv 8 \equiv 2 \pmod{3}$.
So, $n^3 + n \equiv 2 + 2 = 4 \equiv 0 \pmod{3}$.

Since all possible cases give remainder $0$ when divided by $3$, we conclude $3 \mid (n^3 + n)$.

\[
\boxed{\text{Therefore, for all integers } n,\; 3 \mid (n^3 + n).}
\]
