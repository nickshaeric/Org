#+title: Floor & Ceiling
#+filetags: :topic:
* Definition
** Basic Bounds
For all $x \in \mathbb{R}$:
*** Floor
$\lfloor x \rfloor = n \iff n \in \mathbb{Z}$ and $n \leq x < n + 1$.

Notation: $\lfloor x \rfloor =$ the floor of $x$.

*** Ceiling
$\lceil x \rceil = n \iff n \in \mathbb{Z}$ and $n-1 < x \leq n$.

Notation: $\lceil x \rceil =$ the ceiling of $x$.

* Key Points
** Integer Characterization
- $\lfloor x \rfloor = x \iff x \in \mathbb{Z}$.
- $\lceil x \rceil = x \iff x \in \mathbb{Z}$.

* Examples
** $\lfloor \frac{3}{2} \rfloor=1$
Since $1 \in \mathbb{Z}$ and $1 \ge \frac{3}{2} < 1+1$ and $1 \le \frac{3}{2} < 1+1$.

** $\lfloor -\frac{3}{2} \rfloor=-2$
Since $2 \in \mathbb{Z}$ and $-2 \le -\frac{3}{2}<\frac{1}{2}$.

** $\forall x \in \mathbb{R}, \lfloor 2x \rfloor = 2 \lfloor x \rfloor$.

** $\forall x \in \mathbb{R}, \lfloor x^2 \rfloor = \lfloor x \rfloor ^2$.
